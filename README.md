## üöÄ Getting Started

### Prerequisites
* A C Compiler (GCC, Clang, or MSVC).

### How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/enes-ozbilgin/Numerical-Methods.git](https://github.com/enes-ozbilgin/Numerical-Methods.git)
    cd Numerical-Methods
    ```

2.  **Compile the code:**
    ```bash
    gcc Numerical_Methods.c -o numerical_methods -lm
    ```

3.  **Run the executable:**
    ```bash
    ./numerical_methods
    ```
    *(Or simply double-click `Numerical_Methods.exe` on Windows if you trust the build).*
